# KBeerDiscordBot
A simple search and lookup Discord bot for finding information on beer. Provides a name, logo, ABV, IBU, description, and brewery in the current chatroom.

The bot taps into the Discord plugin API using the DSharpPlus (available on Nuget and https://dsharpplus.github.io/index.html) reference and the Untappd API (https://untappd.com/api/dashboard; while a wrapper is available on Nuget, I found it easier to use direct HTTPS requests).
